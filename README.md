# 🤖 Авито AI Ассистент

AI-ассистент для улучшения объявлений на Авито. Сервис анализирует карточку товара и предлагает улучшения текста, а также рекомендует рыночную цену.

## ✨ Возможности

### Основные
- 📋 **Список объявлений** с поиском, фильтрацией и пагинацией
- 👁️ **Просмотр объявления** с полной информацией
- ✏️ **Редактирование** с динамическими полями в зависимости от категории
- 🤖 **AI-генерация описания** через Ollama
- 💰 **AI-рекомендация цены** на основе рыночного анализа
- 💾 **Автосохранение черновиков** в localStorage

### Дополнительные
- 🌓 **Тёмная тема** с сохранением выбора
- 📱 **Адаптивный дизайн** для всех устройств

## 🛠 Технологии

| Технология | Назначение |
|------------|------------|
| **React 18** | UI библиотека |
| **TypeScript** | Типизация |
| **Redux Toolkit** | Глобальное состояние (тема, черновики, фильтры) |
| **React Query** | Серверное состояние и кэширование |
| **Ant Design** | UI компоненты |
| **Axios** | HTTP запросы |
| **Vite** | Сборка и dev сервер |
| **Ollama** | Локальная LLM (llama3) |

## 🚀 Установка и запуск

### Требования

- Node.js 18+
- npm или yarn
- Ollama (для AI функций)

### Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/Hamsike/avito-ai-assistant.git
cd avito-ai-assistant

# Установить все зависимости (сервер и клиент)
npm run install-all

# Настройка порта для сервера
cd server
set PORT=8080   # Windows
# export PORT=8080   # Linux/macOS

# Запустить все сервисы одной командой
cd ..
npm run dev
